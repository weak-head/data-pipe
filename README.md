# Build and Test

Run `make help` for the list of available build commands.

Generate code coverage in form of 'cov' file:
```sh
make coverage
```

Generate code coverage in form of 'html' file:
```sh
make coverhtml
```

Build docker image:
```sh
make build-docker
```
